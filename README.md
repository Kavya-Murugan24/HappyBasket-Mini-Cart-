# HappyBasket Mini Cart

A Java-based mini shopping cart application with:

- User authentication
- Session handling
- Add to cart
- Remove from cart
- Quantity update
- MySQL database integration
- Mini cart functionality

## Technologies Used
- Java
- JDBC
- MySQL
- HTML/CSS
- Session Management

## Features
- Login & Signup
- Cart management
- Order handling
- Dynamic mini cart updates
